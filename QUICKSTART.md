## üì¶ Installation

### Option 1: From GitHub (Recommended)

1. **Download the extension**
   - Click the green "Code" button
   - Select "Download ZIP"
   - Extract the ZIP file

2. **Load in your browser**
   
   **Chrome / Brave / Vivaldi:**
   - Go to `chrome://extensions/`
   - Enable "Developer mode" (top-right toggle)
   - Click "Load unpacked"
   - Select the extracted folder
   
   **Microsoft Edge:**
   - Go to `edge://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the extracted folder
   
   **Opera:**
   - Go to `opera://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the extracted folder

3. **Done!** üéâ

### Option 2: Clone with Git

## üéØ First Use

1. **Click the extension icon** in your Chrome toolbar
2. **You'll see the popup** with a toggle switch
3. **Toggle is ON by default** (green)
4. **Visit ChatGPT** and enjoy ad-free responses!

## üîÑ Toggle Control

### Turn Protection ON
- Click the toggle switch
- It turns **green**
- Status shows "ON"
- Ads will be blocked

### Turn Protection OFF
- Click the toggle switch
- It turns **red**
- Status shows "OFF"
- Ads will not be blocked

## üîß Troubleshooting

### Extension not loading?
- Make sure you selected the correct folder
- Check that all files are present
- Try reloading the extension

### Toggle not working?
- Reload the extension
- Reload the ChatGPT page
- Check browser console for errors

### Not blocking ads?
- Make sure toggle is ON (green)
- Verify you're on chatgpt.com or chat.openai.com
- Check console for initialization message
